# wavpack-rs

Rust bindings for [WavPack](https://www.wavpack.com/). 

The included WavPack version is [5.6.0](https://www.wavpack.com/changelog.txt).

## Dependencies

* [CMake](https://cmake.org) is used to build WavPack.  
* A C++ compiler.
